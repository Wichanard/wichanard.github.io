🛠️ Technical Implementation (ขั้นตอนการปฏิบัติ)
1. SQL Data Profiling & Design
Action: Reviewed existing tables and identified type mismatches.
Result: Updated field properties (e.g., changing VARCHAR to INT where appropriate) to optimize storage and query performance.
ภาษาไทย: ตรวจสอบข้อมูลใน SQL และปรับปรุงประเภทข้อมูลใน Design View ให้เหมาะสม เช่น การเปลี่ยนข้อความให้เป็นตัวเลขเพื่อใช้ในการคำนวณ ช่วยเพิ่มประสิทธิภาพในการ Query



2. Transformation via Power Query
Action: Connected the SQL source to Power Query and performed data type casting.
Result: Handled date/time formatting and decimal precision to ensure no "Data Loss" occurred during migration.
ภาษาไทย: เชื่อมต่อ SQL เข้ากับ Power Query เพื่อแปลงข้อมูล (Data Casting) เช่น การจัดการรูปแบบวันที่และความละเอียดของทศนิยม เพื่อให้ข้อมูลแม่นยำ 100%
